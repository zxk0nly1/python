# pillow之生成p**n hub风格logo 

```powershell
//复制当前目录下的ph.py文件到本地 （或git clone到本地进入pillow目录）
$git clone https://github.com/zxk0nly1/python.git
```

##### 进入pillow目录并安装

```shell
$cd pillow
$pip install pillow=6.2.0
```

#### 使用

```powershell
$ python ph.py Bilibili Hub
```
